Restaurant Manager Application allows to 
create users and have resctricted access to
all pages. Application allows to create 
stock of ingredients and prices, menu items 
with prices, recipies for created menu from 
ingredeints in stock, purchases of created 
menu items with recipies. Purchases make
automatic cost and profit calculation and 
updates data in stock, also limits purchases
if stock is insufficient. Update and delete data.
